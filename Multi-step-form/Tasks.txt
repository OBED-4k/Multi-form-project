Task: Build a Simple Interactive To‑Do List.(1) Requirements
- Create the HTML structure:
- An input field for entering tasks.
- A button to add the task.
- A list (<ul>) where tasks will appear.
- A small <p> element for error messages.
(2)- Use DOM methods in JavaScript:
- Select elements using getElementById, querySelector, and querySelectorAll.
- Add event listeners to the button and form.
- Validate input (e.g., task must be at least 3 characters).
- Append new <li> items to the list dynamically.
- Add a "Delete" button to each task using createElement. - Use removeChild() to delete tasks when the delete button is clicked.
- Use classList to style completed tasks (e.g., toggle a "done" class when clicked). (3)- Extra challenge:
- Replace a task with an "Edited" version using replaceChild().
- Change the color of tasks dynamically with style.
